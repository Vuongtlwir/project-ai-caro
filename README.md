# project-ai-caro

Caro AI (Gomoku) sử dụng thuật toán Minimax với cắt tỉa Alpha-Beta.

## Yêu cầu

- Python 3.8+
- pip

## Cài đặt

```bash
git clone https://github.com/Vuongtlwir/project-ai-caro
```

## Chạy chương trình
### Giao diện người dùng (Pygame)

Chạy file `main.py` để mở giao diện đồ họa:

```bash
python main.py
```
