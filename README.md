# project-ai-caro

Caro AI (Gomoku) sử dụng thuật toán Minimax với cắt tỉa Alpha-Beta.

## Yêu cầu

- Python 3.8+
- pip

## Cài đặt

```bash
pip install -r requirements.txt
```

## Chạy chương trình

### Giao diện người dùng (Pygame)

Chạy file `main.py` để mở giao diện đồ họa:

```bash
python main.py
```

### AI tự đấu (Hard vs Hard)

Chạy file `run_hardvshard.py` để mô phỏng AI tự đấu 10 ván:

```bash
python run_hardvshard.py
```
